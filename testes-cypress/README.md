# Testes técnico com Cypress

**Objetivo:** Criação de Testes Automatizados com Cypress

### Instruções para execução dos testes: ###

#### Pré-requisitos: ####
- Node JS
- NPM

#### Instalar as dependências ####
```
npm install
```

#### Rodar sem interface gráfica ####
```
npx cypress run
```

#### Rodar com interface gráfica ####
```
npx cypress open
```
